🏥 About Lens Care
Lens Care is an advanced web platform that combines entertainment and healthcare solutions for eye care. The project focuses on three key components:

Face Detection with Optical Frame Application – Real-time face detection with virtual frame overlay.
Eye-Improving Games – Engaging games like Spot the Difference and Set to enhance visual acuity.
Eye Hospital Management System – A system to manage patient appointments, records, and resources efficiently.
Our goal is to ensure affordability, rapid access, and expanded reach for quality eye care services.

🚀 Features
📸 Face Detection & Virtual Frames – Try different optical frames in real-time.
🎮 Eye-Improving Games – Fun games to strengthen eye focus and vision.
🏥 Hospital Management System – Appointment scheduling, record management, and resource tracking.
⚡ Fast & Scalable – Built using modern web technologies for a seamless experience.
🛠 Tech Stack
Frontend: HTML, CSS, JavaScript, Next.js
Database: MySQL
Cloud & Hosting: AWS
Backend: Node.js, Express.js
